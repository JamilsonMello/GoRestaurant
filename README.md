# GoRestaurant

## Getting started

1. Clone this repo using clone `git https://github.com/JamilsonMello/GoRestaurant.git`
2. Move yourself to the appropriate directory: `cd GoRestaurant`<br />
3. Run `yarn` to install dependencies<br />
